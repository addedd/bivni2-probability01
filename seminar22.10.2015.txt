3.  Изучили семейство функций для работы с со случайными величинами с известными распределениями
   unif  - равномерное распределение. Случайная величина принимает значение заданного отрезка с одинаковой вероятностью.
   norm(..,mean,sd)  - нормальное распределение (используется тогда когда разброс измеряемого параметра содержит множество  мелких неучтенных факторов) и про Центральной Предельной Теореме к этому распределению стремится среднее посчитанное по случайным выборкам, при стремление размера выборки к бесконечности. 
   binom(..,size,prob) - случайная величина выражающая количество выпадений орла при количестве size подбрасываний монетки с вероятностью  prob выпадения орла.
   pois(.. ,lambda) -  Распределение Пуассона с мат. ожиданием lambda. Случайная величина, распределенное по этому закону выражает количество событий в единицу времени. Например количество посещений веб-страницы в день считается распреденным по этому закону. 
 Последние 2 распределения на семинаре не рассматривали.
   exp(.., rate) --  экспоненциальное распределение. Случайная величина, распределённая по этому закону выражает  промежуток времени между пуассоновскими событиями происходящими с частотой rate.
   t - t-распределение Стьюдента. Распределение  для средних, считаемых по маленьким выборкам значений нормально распределенной величины.  


Для каждого распредения есть четыре функции вида:

r<имя распредения> (например runif(n) )  -- random numbers. Вектор случайных чисел длины n.
p<имя распредения> (например punif(q) )  -- вероятность (probability) того, что x<q.   Куммулятивная функция распредения.
q<имя распредения> (например qunif(p) )   -- Квантиль --  наименьшее значение q, удовлетворяющее неравенству F(q) >= p, где F() -- функция распредения.   Эта функция обратна к функции распределения.
d<имя распредения> (например dunif() )   -- фунция плотности распредения
